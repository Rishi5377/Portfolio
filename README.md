# 👨‍💻 Rishi's Personal Portfolio

Welcome to my personal developer portfolio! This repository contains the source code for my interactive portfolio website built from scratch. It highlights my projects, skills, and experience as a software developer, offering a clean, responsive, and engaging user experience.

---

## 🌐 Live Demo

You can view the live, deployed version of this portfolio here:
 [Portfolio ](https://rishi5377.github.io/Portfolio/)

---

## 🎯 Features

- **Modern UI/UX:** Clean, elegant, and modern design.
- **Fully Responsive:** Adapts beautifully across desktops, tablets, and mobile devices.
- **Interactive Animations:** Smooth scrolling, hover effects, and dynamic element transitions.
- **Project Showcase:** A dedicated section to display my recent works and collaborations.
- **Skill Highlights:** A clear breakdown of my technical stack and proficiencies.
- **Contact Form:** An easy way for visitors, recruiters, and collaborators to get in touch.

---

## 🛠️ Built With

This portfolio is built entirely with core web technologies, ensuring maximum performance and full customization without relying on heavy frameworks:

- **HTML5** - For the structure and semantics of the content.
- **CSS3** - For styling, layouts, and responsive design (Flexbox/Grid).
- **JavaScript (Vanilla)** - For interactivity, DOM manipulation, and dynamic animations.

## 🚀 Getting Started

If you want to run this project locally or use it as a base for your own portfolio, follow these simple steps:

### Prerequisites
You only need a modern web browser. No complex build tools or package managers required!

### Installation
1. **Clone the repository:**
   ```bash
   git clone https://github.com/Rishi5377/Portfolio.git
   ```
2. **Navigate into the directory:**
   ```bash
   cd Portfolio
   ```
3. **Run the project:**
   Simply open `index.html` in your favorite web browser!
   *(Tip: Use the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension in VS Code for a better development experience).*
